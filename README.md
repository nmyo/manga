# JM Boom - Android

禁漫天堂第三方 Android 客户端。

基于 [ppxb/jm-boom](https://github.com/ppxb/jm-boom) 移植。

## 构建

```bash
pnpm install
pnpm tauri android init
pnpm tauri android build
```

## 特性

- 首页/每周推荐/搜索/详情/收藏/阅读历史
- 章节下载/批量下载
- 自动阅读模式
- 多种阅读模式（单页/条漫、左右/右左翻页）

## 下载

前往 [Actions](https://github.com/nmyo/manga/actions) 下载最新构建的 APK。
